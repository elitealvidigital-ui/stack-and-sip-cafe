# Stack & Sip Cafe

A cinematic gourmet burger and specialty coffee cafe concept built with React, Vite, GSAP ScrollTrigger, and an optimized WebP frame sequence.

## Highlights

- Scroll-controlled burger assembly story
- Responsive interactive menu and order bag
- Reservation flow, cafe story, reviews, hours, and directions
- Desktop and mobile motion treatment with reduced-motion support

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

This is a fictional concept cafe. Business claims and contact details must be replaced before commercial launch.
